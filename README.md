# Mastery Progress

Mastery Progress is an elite, professional study tracker designed for individuals pursuing excellence in high-stakes fields. It combines a prestigious aesthetic with powerful analytical tools to help you achieve mastery in any subject.

## Features

- **Elite Dashboard**: A high-contrast, premium interface providing a comprehensive overview of your progress, readiness score, study streaks, and total hours.
- **Mastery Tracker**: Deeply monitor your understanding of complex topics and sub-topics with weighted mastery calculations.
- **Study Calendar**: Visualize your commitments, set milestones, and strategically plan recovery days to avoid burnout.
- **Mistake Journal**: Log and analyze errors to identify patterns in your reasoning and fix them before they become habits.
- **Session Logger**: Detailed logging of study sessions to track focus and time allocation across different topics.
- **Financial Calculator**: A built-in, sophisticated financial toolset for quick calculations without leaving your study environment.
- **PWA Ready**: Installable on mobile and desktop for a native-app experience with offline support.

## Tech Stack

- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)

## Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd mastery-progress
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create a production-ready build:

```bash
npm run build
```

The output will be in the `dist` directory.
